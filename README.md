# HabbitaMobile Nativo

Aplicativo Android nativo para listagem e busca de imóveis.
Projeto do PI do 5º semestre de DSM.

## Tecnologias

- Java
- Android SDK 14
- RecyclerView, CardView
- Biometria

## Funcionalidades

- Listagem de imóveis com cards
- Filtros por tipo, garagem, área mínima e máxima
- Tela de detalhe do imóvel
- Favoritar imóveis
- Contato com anunciante (WhatsApp, ligação, e-mail)
- Autenticação com biometria
