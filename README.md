# 🏠 HabbitaMobile Nativo

![Android](https://img.shields.io/badge/Android-SDK%2014-green?logo=android)
![Java](https://img.shields.io/badge/Java-orange?logo=java)
![Status](https://img.shields.io/badge/Status-Em%20desenvolvimento-yellow)

Aplicativo Android nativo para listagem e busca de imóveis.
Projeto Integrador do 5º semestre de DSM.

---

## 📱 Funcionalidades

- 🏘️ Listagem de imóveis com cards
- 🔍 Filtros por tipo, garagem, área mínima e máxima
- 📄 Tela de detalhe do imóvel
- ❤️ Favoritar imóveis
- 📞 Contato com anunciante (WhatsApp, ligação, e-mail)
- 🔐 Autenticação com biometria

---

## 🛠️ Tecnologias

- Java
- Android SDK 14
- RecyclerView, CardView
- BiometricPrompt API
